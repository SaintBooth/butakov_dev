## Проблема

Сборка падала с ошибкой ERESOLVE из-за конфликта версий бандлера и библиотеки стилей. При деплое сервер уходил в бесконечный цикл редиректов — маршрутизатор облачного провайдера принимал HTTPS-запрос, передавал его по внутренней сети в конфиг, который снова редиректил на HTTPS. Система аналитики предупреждала о перехвате функций ввода из-за контролируемых UI-компонентов.

## Решение

Принудительная установка зависимостей в обход строгой проверки версий. Переписал конфиг сервера для корректной отдачи статики. Очистил кеш локальной сети, чтобы убрать фантомные ошибки сертификата. Добавил блокирующий атрибут в разметку формы для совместимости с аналитикой.

Исправление конфига сервера для цикла редиректов:

```apache
<IfModule mod_rewrite.c>
RewriteEngine On
RewriteBase /
RewriteRule ^index.html$ - [L]
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule . /index.html [L]
</IfModule>
```

Обход конфликта пакетного менеджера:

```bash
npm install -D tailwindcss @tailwindcss/vite --legacy-peer-deps
```

Изоляция аналитики от полей ввода:

```html
<form data-ym-disable-keys="true">
```
